# 🔭 天文攝影查詢 Bot

台灣天文攝影專用 LINE Bot，結合天文計算、氣象預報與 AI 自然語言回覆。

產品定位不是「只服務銀河攝影」，而是協助台灣攝影者判斷各類天文與天空景象是否值得出勤：

- 深空與星野：銀河、星座、星雲、星系、流星雨、彗星等
- 日月行星運行景象：月出月落、月相、日月方位、行星與月亮接近、行星可見性等
- 天氣與地景條件：雲量、透明度、視寧度、結露、霧、雲海、海岸潮汐等
- 攝影決策：去哪裡、何時拍、風險是什麼、備案地點與器材提醒

---

## 使用方式

掃描 LINE Bot QR Code 加入好友，直接用中文輸入查詢：

```
4月15日 合歡山 銀河
這個週末 阿里山 有什麼可以拍？
5月1日到3日 墾丁 天蠍座
今晚 外澳 適合拍攝嗎？
```

## 功能說明（v2 + Phase 3A）

### ✅ 查詢功能

| 功能 | 說明 |
| --- | --- |
| 自然語言查詢 | 直接用中文輸入，不需要特定格式 |
| 指定標的查詢（類型 A） | 指定天體，回傳最佳觀測時刻與條件 |
| 開放探索查詢（類型 B） | 不指定天體，Bot 列出當晚所有可拍標的 |
| 日期區間查詢 | 支援單日、多日、「今晚」「這個週末」等表達方式 |
| 地點解析 | 先查 113 個已審核地點；查不到時反問座標，不讓 AI 亂猜 |
| 地點比較模式 | 支援「合歡山 vs 阿里山 這週末」語氣，兩地 CCI 並排比較 |
| 最佳地點排名 | 「今晚哪裡最好拍」→ 從全台 approved 地點排出前 6 名；支援指定北/中/南/東/離島區域 |
| 15 天氣象日曆 | 輸入地點，顯示未來 15 天每晚天況評估 |

### ✅ 出勤信心指數（CCI）

每晚 0–100 分，純 Python 計算，不依賴 AI 推測。

| 分數 | 判斷 | 說明 |
| --- | --- | --- |
| ≥ 80 | ✅ 強烈推薦 | 各項條件皆佳 |
| ≥ 60 | 🟢 值得出勤 | 主要條件良好 |
| ≥ 40 | ⚠️ 謹慎考慮 | 有明顯風險因子 |
| ≥ 20 | 🟠 不建議 | 條件較差 |
| < 20 | ❌ 不值得出勤 | 出勤效益極低 |

六個加權因子：雲量 30%、有效暗空窗口 22%、視寧度 13%、透明度 10%、目標可見性 10%、風速 10%、結露風險 5%

每次回覆同時列出「最強不去理由」（反樂觀守則），防止 AI 過度樂觀。

### ✅ 標的庫（14 個天體）

| 類型 | 標的 |
| --- | --- |
| 銀河 | 銀河核心 |
| 星座 | 獵戶座、天蠍座、獅子座、仙女座、南十字座 |
| 星雲／星系 | 獵戶座大星雲 M42、礁湖星雲 M8、鷹星雲 M16、仙女座星系 M31、玫瑰星雲 NGC2244、猴頭星雲 NGC2174、昆蟲星雲 NGC6302 |
| 彗星 | 紫金山-ATLAS（僅提供氣象 CCI，不提供即時位置與方位角） |
| 流星雨 | 象限儀、英仙、雙子、獅子、天琴（極大期 ±3 天提示） |

### ✅ 計算與預報

| 功能 | 說明 |
| --- | --- |
| 天文計算 | Skyfield 每 10 分鐘精確計算仰角與方位角，每天取最佳時刻 |
| 月相分析 | 顯示每日月相及對深空攝影的影響程度 |
| 月出月落 | 計算月出月落時刻與方位角，標示整夜月上/月下情況 |
| 有效暗空窗口 | 以天文薄暮（太陽低於 -18°）為基準，剔除月光時段，計算真實可用拍攝窗口 |
| 銀河構圖方位 | 計算銀河核心方位角與中文方向描述，結合月亮方位評估構圖干擾 |
| 氣象預報 | Open-Meteo 提供雲量、濕度、溫度、露點、風速（未來 15 天）|
| 結露風險 | 溫度與露點差 < 1.5°C 時自動提醒 |
| 曝光建議 | 500 法則快門、依月相分段 ISO、光圈建議 |

### ✅ Bot 操作

| 功能 | 說明 |
| --- | --- |
| 服務選單 | 輸入「選單」或 `/menu` 叫出快捷功能列 |
| 取消查詢 | 計算中可按「❌ 取消」中止 |
| 用戶評分 | 每次回覆後顯示 👍 👎 按鈕，記錄氣象準確度 |
| 許願池 | 按 💡 可輸入建議，自動記錄至 Google Sheets |
| 多用戶隔離 | 每位用戶對話完全獨立 |
| 自動重啟 | 遇到網路錯誤自動重啟 |

## 目前不支援

### ❌ 不支援的攝影題材

| 題材 | 原因 | 目前應對方式 |
| --- | --- | --- |
| 日落 / 日出 | 氣象邏輯與天文攝影相反（適量雲彩反而是加分）；需要彩霞預測模型 | Bot 提供基礎天況，不給 CCI 評分 |
| 懸日 | 除氣象外需要精確地景方位角對齊計算 | 攔截，說明尚未支援 |
| 雲海 | 需垂直濕度梯度與逆溫層資料；現有氣象框架無法支撐 | 攔截，說明尚未支援 |
| 日月行星可見性 | 行星即時位置計算尚未整合 | 攔截，說明尚未支援 |
| 彗星即時位置 | 需 JPL Horizons 即時軌道座標；靜態座標精度不足，不提供方位建議 | 提供氣象 CCI，位置請自行查詢 |
| 極光 | 台灣緯度幾乎不可見 | 攔截，說明地理限制 |

### ⚠️ 已知限制

| 項目 | 說明 |
| --- | --- |
| 氣象預報範圍 | 僅支援未來 15 天；超出範圍仍顯示天文計算，但無氣象資料 |
| 微氣候修正 | 尚未建立各地點的歷史預報誤差修正模型；山區微氣候可能與預報有落差 |
| 回應速度 | 每次查詢約 30–60 秒；正在優化中（Phase 3B 目標 < 15 秒） |
| 無地點記憶 | 每次查詢都需要輸入地點 |
| 無主動推播 | 目前只能被動回應，不會主動通知天況好的夜晚（Phase 5 規劃中）|

## 技術架構

```
LINE 訊息（Webhook → Flask）
    ↓
意圖解析（OpenRouter API）
    ↓
天文計算（Skyfield + de421.bsp）
    ↓ 月出月落、天文薄暮、暗空窗口、銀河構圖方位角
氣象預報（Open-Meteo API）
    ↓
回覆生成（OpenRouter API）
    ↓
反饋記錄（Google Sheets API）
```

部署於 Render（Web Service），Flask 接收 LINE Webhook，24 小時持續運行。
